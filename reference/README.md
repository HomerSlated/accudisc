# reference/

Local-only staging area for source trees used as analysis references. Contents
are **git-ignored** (except this file) — link or copy trees in freely:

- `c2read` — the original prototype this project grows from
- Redumper, cdrdao, readcd/cdrecord (cdrtools), cd-paranoia (libcdio-paranoia)
- CUETools / CUERipper
- Reverse-engineering material: firmware, vendor tools, protocol notes

Code here is read for understanding, then **rewritten** for AccuDisc — not
copied — since several of these trees are GPL/CDDL and AccuDisc must keep a
clean licensing story.
